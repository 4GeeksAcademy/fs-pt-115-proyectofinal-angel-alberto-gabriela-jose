
# AURA 


Menos discusiones, más recompensas.

AURA es una aplicación que gamifica la gestión de tareas compartidas para parejas, compañeros de piso y equipos.

La mecánica es simple: completa tareas, gana puntos y canjéalos por premios personalizados que tu propio grupo define (ej: "una cena fuera", "elegir la próxima serie", etc.).

Nuestro objetivo es transformar la obligación en motivación, promoviendo un reparto justo de responsabilidades y reduciendo los conflictos a través de un sistema transparente, colaborativo y, sobre todo, divertido.



## Installation

Install my-project with npm

```bash
  instala los paquetes: npm install 
  inicia el frontend : npm run start
  inicia el backend : pipenv run start
  
```
